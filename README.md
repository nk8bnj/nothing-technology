<h2 align="center">Running application locally</h2>

To get started:

1. Clone the repository to your local machine:

```
git clone https://github.com/nk8bnj/nothing-technology
```

2. Install dependencies using Node 14:

```
npm install
```

3. Launch the project:

```
npm start
```
